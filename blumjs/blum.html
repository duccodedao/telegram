Thiết lập Blum Auto Clicker

  1. Cài đặt Ignore X-Frame Headers.
  2. Cài đặt Tampermonkey.
  3. Tải xuống file script: Tải blum.js
  4. Thêm script vào Tampermonkey:
    1. Mở Tampermonkey.
    2. Nhấn vào "Thêm script mới".
    3. Sao chép và dán mã từ file `blum.js` đã tải về vào trình soạn thảo script.
    4. Lưu lại script.

Sau khi thiết lập, mở Blum và để script chạy.
