Thiết lập Blum Auto Clicker

  1. Cài đặt Ignore X-Frame Headers.
  2. Cài đặt Tampermonkey.
  3. Coppy Blum.
  4. Thêm script vào Tampermonkey:
    1. Mở file blum.js dưới dạng Notepad
    2. Copy toàn bộ mã js
    3. Mở Tampermonkey.
    4. Nhấn vào "Thêm script mới".
    5. Sao chép và dán mã từ file `blum.js` đã tải về vào trình soạn thảo script.
    6. Lưu lại script bằng cách nhấn vào chữ "Tệp" và "Lưu"

Sau khi thiết lập, mở Blum và để script chạy.
